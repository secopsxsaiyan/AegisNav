### 2. 🟡 WARNING — Exported Activity
- **Rule:** `java.android.security.exported_activity.exported_activity`
- **File:** `app/src/main/AndroidManifest.xml:64`
- **Description:** MainActivity is declared with `android:exported="true"`.
- **Code:**
  ```xml
  <activity
      android:name=".MainActivity"
      android:exported="true"
      android:windowSoftInputMode="adjustResize"
      android:theme="@style/Theme.AegisNav"
      tools:targetApi="31">
      <intent-filter>
          <action android:name="android.intent.action.MAIN" />
          <category android:name="android.intent.category.LAUNCHER" />
      </intent-filter>
  </activity>
  ```
- **Assessment: ✅ FALSE POSITIVE (required by Android)**
  - `android:exported="true"` is **mandatory** for any activity with an `intent-filter`. Android 12+ (API 31+) requires this to be explicitly set — the build would fail without it.
  - This is the app's launcher activity. It *must* be launchable by the OS home screen launcher.
  - No sensitive operations or privileged data are exposed solely by launching MainActivity.
  - The `ScanService` is correctly set to `android:exported="false"`.
